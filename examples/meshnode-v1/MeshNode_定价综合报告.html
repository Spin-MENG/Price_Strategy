上市路线图 · 结论与行动

把分析变成动作，并明确什么数据需要在上市后回流校准。

3 个核心结论

  1. 四种独立方法收敛在 €194-€220/3-pack 区间（中位 €205），€73/节点处于下沿，对用户友好且利润健康。
  2. 价格敏感度低 — 即使涨到 €90，市场购买意愿率仅从 30.5% → 24.7%，证明有进一步定价空间。
  3. 最大未满足缺口在 C 段（€150-220）的 Mesh 稳定性 + 覆盖盲区，本品规格能正面攻击。

4 个下阶段行动

  1. 三轨 SKU 同步上市：攻击轨 €65×3 / 主力轨 €73×1 / 溢价轨 €90×1（先 DE 市场打透，再外溢）。
  2. 埋点真实销售数据：attach rate / 单 SKU 销量 / 实际成交价 → 回流校准购买意愿率模型。
  3. C 段进攻沟通："稳定性 + 长固件支持 + 无订阅墙 + OpenWrt"作为对 Nest/FRITZ 的攻击文案主轴。
  4. 3 个月后复盘：用真实成交数据反推（数据反推），更新基准价 v2。
