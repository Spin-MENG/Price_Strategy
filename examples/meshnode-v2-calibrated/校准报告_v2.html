三句话总结： （1）v1 模型的 12 个原信号绝大多数（9 个）经住校准，cluster-aggregated proxy 不是凭空捏造。 （2）但 v1 加的 5 个新「差异化武器」里有 3 个被发现是过度乐观——订阅厌恶 / 无订阅价值 / 回程失败在真实购买评论里只有 0.7%-2.1% 提及率， 因为这些声音主要发生在 Reddit 讨论而非 Amazon 购买后评论。 （3）最被低估的真实信号是「退货意图」——15.2% 评论明确表达「退货 / 不会再买」， 退货池规模约 480 条评论，远大于 v1 报告说的「24 条明确证据」。
对定价决策的影响：€73 主力价 + €285 套装 仍是最优（基于第四法 + 痛点对位，与 P(buy) 绝对值无关）。 但落地页 hero 不应再单押「不收订阅」—— 改为复合 hero：「30 天无理由退（攻击退货池）+ OpenWrt/VPN/VLAN 全开放（pro 用户）+ 3 年 TCO 省 €257（精准投放 eero/AVM 受害者）」。
校准方法

用 LLM 跑全量代替人工 N=200 · 25 分钟完成 · 比人工快 200 倍

原 Phase 1.5 计划：抽 N=200 条 stratified by 18 segments，两位 reviewer 独立标 17 信号 0/1/2，算 Cohen's κ + F1。 人手成本太高（≥ 2 周）+ 上市档期不等人，改用 LLM 自动代理：让 DeepSeek-Chat 对全部 3,392 条评论按相同 17 信号定义打分。

17 信号校准 · 按相关性排序

相关性 ≥ 0.5 = 可信 · 0.3-0.5 = 灰色 · < 0.1 = 无效 · 新信号无对照（n/a）按 LLM 提及频率判等级

4 个最被验证的核心信号（相关性 > 0.85）： GL.iNet 老用户身份 (0.997) · OpenWrt 使用 (0.989) · VPN 用例 (0.942) · 稳定性痛点 (0.860)。 这 4 个是 v1 模型最重要的差异化拉力，全部经住校准——意味着「MeshNode 应该主打 OpenWrt + VPN + 稳定性 + 老客户加购」的定位完全成立。
1 个信号被发现完全无效：setup_friction（相关性 0.078）—— v0 提及率 55.6% 因为「setup」一词太通用，校准证明该信号几乎无相关性。 从模型剔除（权重归零），与 v1 报告预判一致。
5 新信号

v1 新加的 5 个差异化武器 · 哪些过度乐观

v1 模型在 12 个原信号上加了 5 个「差异化武器」（订阅厌恶 / 回程失败 / 退货意图 / 无订阅价值 / Pro 控制），权重来自 agent 萃取的定性发现。 校准用 LLM 在真实评论里的提及频率验证：高频 = 武器活跃，低频 = 武器哑火。

5 个新信号 · LLM 在 3,392 条评论中的提及频率

绿色 = 提及 ≥ 5% 武器活跃 · 金色 = 1-5% 偏弱 · 红色 = < 1% 武器哑火

退货意图（15.2%）+ Pro 控制（13.1%）是真正活跃的差异化信号—— v1 模型对这两个的权重设定（+1.50 / +0.80）经住校准。 退货池规模约 480 条评论，是 v1 报告说的「24 条明确证据」的 20 倍， 意味着「30 天无理由退 + 同价更稳」是 mass-market 可投放的核心 hero。
3 个新信号被发现是过度乐观： 订阅厌恶 2.1%（Amazon 评论里很少明确抱怨订阅墙，这个声音主要在 Reddit 讨论）、 回程失败 2.0%（被合并到「稳定性」笼统讲，单独提的少）、 无订阅价值 0.7%（几乎没人主动表扬「免订阅」）。
校准动作：3 个信号权重折让 40-70%， 「3 年省 €257 vs eero」改为精准投放（已用 eero/AVM 的失望用户）而非 mass-market hero。

决策影响 + 行动

4 个核心结论 + 4 个落地行动

校准后的定价 + 营销决策清单。

4 个核心结论

  1. 定价决策不变：€73 主力 + €285 套装仍是最优组合（基于第四法 + 三法中位，与购买意愿绝对值无关）。
  2. OpenWrt / VPN / 稳定性 / 覆盖 4 个高置信信号全部经住校准（相关性 0.80-0.99），是 MeshNode 真正的差异化护城河。
  3. 「订阅厌恶」hero 被高估——真实购买评论里只有 2.1% 提及，不适合作 mass-market hero；退货意图 15.2% 才是被低估的真实信号。
  4. 校准后购买意愿下调 10pp，主力价 27.5% 是更可靠的上市预期，可直接作为 Phase 4 真实销售的对照基线。

4 个落地行动

  1. 调整落地页 hero 优先级：第一 hero 改为「30 天无理由退 + 同价更稳」（攻击 480 条退货意图评论代表的人群）。
  2. 「3 年省 €257」从 mass-market 改为精准投放：只对真实使用 eero/AVM 的用户（GoodCloud 注册时识别）推该 hero。
  3. OpenWrt + VPN + VLAN 全开放作为第二 hero（覆盖 pro_control 13.1% + openwrt 2.8% + vpn 5.1% 三个高置信信号）。
  4. setup_friction 信号从模型剔除，相关脚本和报告下游全部去除该字段；不再用「上手简单」做营销词（v0 报告也说过该信号过宽）。

报告里几个不可避免的术语

LLM 自动代理标注
用大语言模型（DeepSeek-Chat）代替人工 reviewer 对评论按信号定义打分；用全量代替抽样，成本几乎为零（< $2）
相关性（Pearson r）
LLM 评分与原模型估计的统计一致性；范围 -1 ~ +1；> 0.5 = 高度一致；< 0.1 = 几乎不相关
购买信号 / 权重
「该用户的某个特征出现时，会让购买意愿上升或下降多少」的量化估计；模型用 17 个信号 + 17 个权重组合预测购买意愿
购买意愿率
该用户分群在该价位上「会购买」的概率估算（模型推算，不是实际销售数据）
分群（segment）
按用户行为特征 AI 聚类出的具有相似购买动机的人群
退货意图
评论中明确表达「退货 / 不会再买 / 准备换牌」的信号；mass-market 营销最敏感的人群
Phase 4 真实销售校准
上市后用真实订单数据反推模型参数；是本次校准之后的最终 ground truth
